# backnd

A polished, minimal D2C management dashboard for brand owners. This first version is a static frontend, so it can be deployed to Vercel immediately and connected to real data later.

## Run locally

```bash
npm start
```

Then open `http://localhost:4174`.

## Deploy on Vercel

Use the default static deployment. No build command is required.

Recommended settings:

- Framework preset: Other
- Build command: leave empty
- Output directory: `.`

## What is included

- Responsive dashboard shell for desktop, tablet, and mobile
- Sidebar sections for platforms, inventory, logistics, ads, customers, finance, and support
- Accessible landmarks, labels, focus states, and reduced-motion support
- Static sample metrics ready to replace with API data
